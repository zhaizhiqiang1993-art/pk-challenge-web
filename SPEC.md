# 练习网站规格文档

## 1. 项目概述

- **项目名称**：答题练习网站
- **项目类型**：教育类 Web 应用
- **核心功能**：提供数学、语文、英语三种学科的答题练习
- **目标用户**：学生群体（小学生为主）

## 2. 模块划分

### 2.1 口算PK（数学）
| 游戏类型 | 说明 |
|---------|------|
| 比大小 | 比较两个数/算式的大小，选择 > < = |
| 简单加减法 | 10/20以内加减法，4个选项选择正确答案 |
| 表内乘除法 | 1-9乘法口诀范围的乘除法 |

### 2.2 语文PK（语文）
| 游戏类型 | 说明 |
|---------|------|
| 成语填词 | 看图或描述，填写正确的成语 |
| 诗词补 | 给出上句填下句，或给出下句填上句 |

### 2.3 单词PK（英语）
| 游戏类型 | 说明 |
|---------|------|
| 单词对对碰 | 英文单词与中文意思连线匹配 |
| 听读音选单词 | 播放英语单词发音，选择正确的单词 |

## 3. UI/UX 规格

### 3.1 布局结构
```
┌─────────────────────────────────────┐
│           网站标题/导航             │
├─────────────────────────────────────┤
│                                     │
│         三个模块选择卡片            │
│    ┌─────┐  ┌─────┐  ┌─────┐      │
│    │数学 │  │语文 │  │英语 │      │
│    └─────┘  └─────┘  └─────┘      │
│                                     │
├─────────────────────────────────────┤
│           游戏答题区域               │
│                                     │
├─────────────────────────────────────┤
│      得分 | 进度 | 计时器           │
└─────────────────────────────────────┘
```

### 3.2 配色方案
- 主色：#4F46E5（靛蓝色）
- 次色：#10B981（绿色-正确）
- 错误色：#EF4444（红色）
- 背景色：#F8FAFC（浅灰白）
- 卡片背景：#FFFFFF

### 3.3 字体
- 标题：'Noto Sans SC', sans-serif
- 数字/英文：'JetBrains Mono', monospace
- 标题：28px
- 正文：16px

### 3.4 组件
- 选择卡片：圆角16px，悬停上浮+阴影
- 答题按钮：圆角12px，选中高亮
- 进度条：圆角，渐变填充
- 正确/错误反馈：颜色变化+动画

## 4. 功能规格

### 4.1 通用功能
- 每轮 10 道题
- 计时显示
- 实时得分
- 答题进度显示
- 结束后显示正确率和用时

### 4.2 口算PK
- **比大小**：显示两个数字或算式，选择 >
- **加减法**：随机生成10/20以内算式
- **乘除法**：1-9口诀范围

### 4.3 语文PK
- **成语填词**：显示成语含义/图片，填写缺失字
- **诗词补**：经典诗句填空

### 4.4 单词PK
- **单词对对碰**：左右两组词汇，连线匹配
- **听读音选单词**：使用 Web Speech API 播放发音

## 5. 验收标准

- [ ] 三个学科模块可正常切换
- [ ] 口算三种题型可正常答题
- [ ] 语文两种题型可正常答题
- [ ] 单词两种题型可正常答题
- [ ] 计时和计分功能正常
- [ ] 答题反馈正确/错误清晰
- [ ] 响应式适配移动端
